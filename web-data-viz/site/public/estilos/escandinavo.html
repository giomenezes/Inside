Estilo Escandinavo

O estilo Escandinavo busca misturar o moderno com o tradicional. Uma boa maneira para fazer isso é investir nas tonalidades claras para conmpor o visual dos ambientes que irá decorar. É destacado para aconchego e conforto em todos os detalhes.

Características:

  • Funcional;
  • Simples;
  • Estiloso.

    • Elementos chave:

  • Linhas retas;
  • Cores neutras;
  • Luz natural;
  • Ambientes integrados;
  • Materiais naturais.

    • Cores&Materiais:

    Cores como branco, cinza e bege são ótimas para servir como "pano de fundo nesse estilo.

    Móveis elegantes feitos, sobretudo, de madeira. Combina móveis com madeira e ferro. Dá destaque ao minimalismo. E faz uso de acessórios modernos.
